## GRPC with golang

### Quick start
- https://grpc.io/docs/languages/go/quickstart/

```
    go get -u google.golang.org/grpc
    go get -u github.com/golang/protobuf/protoc-gen-go
```

### How to compile & run 
- make greet
#### Build & Run server
- ./bin/greet/server
#### Build & Run client
- ./bin/greet/client
